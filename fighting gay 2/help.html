HELP c'est par là :

Joueur 1 Controls:

  • a = Se déplacer à gauche
  • d = Se déplacer vers la droite
  • w = Sauter
  • s/Spacebar = RAAAttack
  • e = Dommage émotionnel


    • Remarque : Assurez-vous de désactiver le verrouillage des majuscules



Joueur 2 Controls:

  • ArrowLeft Key = Se déplacer à gauche
  • ArrowRight Key = Se déplacer vers la droite
  • ArrowUp Key = Sauter
  • ArrowDown Key = RAAAttack
  • Enter Key = Dommage émotionnel


Note: Les dégâts émotionnels ne peuvent être utilisés qu'une seule fois par joueur dans un match.
